Zmodem.js Integration

To enable full Zmodem support, you need to:

Option 1: Use zmodem.js Library (Recommended)

  1. Download zmodem.js from: https://github.com/FGasper/zmodemjs
  2. Include the library in index.html
  3. The library will handle the full Zmodem protocol in the browser

Option 2: Server-side lrzsz

  1. Install lrzsz: sudo apt-get install lrzsz
  2. Pipe the connection through rz/sz commands
  3. More complex but handles all edge cases

Option 3: Alternative Transfer Methods

  • Many BBSes offer HTTP/FTP downloads
  • Use a dedicated BBS client like SyncTERM
  • Some BBSes support Kermit or Xmodem which are simpler

The current implementation detects Zmodem but doesn't handle the full protocol.
